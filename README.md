# Tugas Android Studio - Week 5 (Form Registrasi)

Proyek ini adalah aplikasi Form Registrasi yang mengimplementasikan berbagai komponen UI dan logika validasi sesuai instruksi tugas.

## Fitur yang Diimplementasikan:
1. **Complete Form (Poin 01):** Input Nama Lengkap, Email, Password, dan Confirm Password.
2. **Advanced Validation (Poin 02):** Validasi format email dan pengecekan kecocokan password secara real-time.
3. **Selection Controls (Poin 03):** Pemilihan jenis kelamin (RadioButton) dan hobi (Checkbox) dengan minimal 3 pilihan.
4. **Spinner & Dialog (Poin 04):** Pemilihan kota asal melalui Spinner dan Dialog konfirmasi sebelum pengiriman data.
5. **Gesture Interaction (Poin 05):** Fitur Long Press pada tombol daftar untuk menyimpan data sebagai draft.
6. **GitHub Repository (Poin 06):** Source code dikelola dan diunggah ke GitHub.

## Cara Menjalankan
1. Clone repository ini.
2. Buka di Android Studio (versi terbaru disarankan).
3. Jalankan pada emulator dengan API level 30+.
