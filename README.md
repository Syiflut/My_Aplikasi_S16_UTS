## 🚀 Fitur Utama
Aplikasi ini mencakup beberapa poin penilaian utama sesuai instruksi tugas:

1. **Sistem Login Sederhana:** Menggunakan autentikasi hardcode (Email: `admin@gmail.com`, Pass: `12345`).
2. **Dashboard Utama:** Halaman sambutan setelah login berhasil dengan akses ke form pendaftaran.
3. **Form Pendaftaran Lengkap:** Input data menggunakan widget EditText, RadioButton (Jenis Kelamin), Spinner (Pilihan Seminar), dan Checkbox (Persetujuan).
4. **Validasi Input Real-time:**
   * Nama dan Email wajib diisi (Email harus mengandung "@").
   * Nomor HP wajib diawali "08", terdiri dari 10-13 digit, dan hanya berupa angka.
   * Validasi Checkbox sebelum data dikirim.
5. **Dialog Konfirmasi:** Muncul sebelum berpindah ke halaman hasil untuk memastikan kebenaran data.
6. **Halaman Hasil (Output):** Menampilkan ringkasan data yang telah diinputkan oleh pengguna.
